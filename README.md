# 2021nlpi

#### 个人作业
1. 编写一个爬虫程序，爬取语料，可参考已上传的例程。程序推荐格式为`.py`文件。
2. 爬取对象（如例程已选择光明网、头条、起点中文网）不唯一，可在`爬取网站列表.xlsx`中选取。
3. 爬取规模大概在500篇（新闻、章节、帖子等）。若选择的爬取对象与例程相同，则爬取的内容应与例程不同。
4. 完成后，请将个人爬虫程序与爬取的语料打包压缩发送给助教。助教邮箱：`mengzhaotai@pku.edu.cn`
5. 提交时间初步定为5月份
